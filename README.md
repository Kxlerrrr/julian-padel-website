# 🎾 Julian Padel Academy - Interactieve Website

Dit is de complete, modern ontworpen single-page webapplicatie voor de **Julian Padel Academy**. De website is volledig responsive, uitgerust met automatische donkere modus en bevat een dynamisch winkelwagensysteem en spelersdashboard gebouwd in **Vue 3** en **Tailwind CSS**.

## 🚀 Live Testen op GitHub Pages

Volg deze stappen om de website binnen 1 minuut gratis online te publiceren via GitHub:

1. Maak een nieuwe repository aan op GitHub (bijvoorbeeld genaamd `julian-padel-academy`).
2. Upload alle bestanden uit deze map (`index.html`, `app.js`, etc.) naar de repository.
3. Ga op GitHub naar de instellingen van je repository (**Settings**).
4. Klik in het linkermenu op **Pages** (onder de sectie *Code and automation*).
5. Kies bij **Source** voor *Deploy from a branch*.
6. Selecteer onder **Branch** de `main` (of `master`) branch en klik op **Save**.
7. Na ongeveer een minuut genereert GitHub een live link (bijv. `https://jouwgebruikersnaam.github.io/julian-padel-academy/`).

## ✨ Functionaliteiten

* 🛒 **Multi-booking Systeem**: Voeg flexibel meerdere lessen toe aan een dynamische winkelwagen en reken alles in één keer virtueel af.
* 🔒 **2FA Inlogsysteem & Dashboard**: Simuleer een volledige inlogketen met tweestapsverificatie. Na inloggen opent direct het spelersdashboard met resterende rittenkaartsaldovariabelen.
* 🌓 **Tailwind Dark Mode**: De website start direct in een rustige, moderne dark-mode setting.
* 🔄 **Trustpilot API Simulator**: Een geautomatiseerde review-carrousel die soepel roteert tussen klantbeoordelingen.
